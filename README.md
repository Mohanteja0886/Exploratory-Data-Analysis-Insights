# World Happiness Report Analysis (2019)

## Project Overview

This project performs an Exploratory Data Analysis (EDA) on the World Happiness Report 2019 dataset. The objective is to understand the factors that influence happiness across different countries and identify meaningful patterns through statistical analysis and data visualization.

## Dataset

Source: World Happiness Report 2019

The dataset contains country-level happiness scores and several contributing factors such as:

* GDP per Capita
* Social Support
* Healthy Life Expectancy
* Freedom to Make Life Choices
* Generosity
* Perceptions of Corruption

## Objectives

The analysis answers the following questions:

1. What are the top 10 happiest countries?
2. What are the bottom 10 happiest countries?
3. What is the average global happiness score?
4. Which country has the highest GDP per capita?
5. What is the correlation between GDP and happiness score?

## Data Cleaning

The following preprocessing steps were performed:

* Checked dataset structure and data types
* Identified missing values
* Removed duplicate records
* Verified dataset consistency
* Prepared clean data for analysis

## Technologies Used

* Python
* Pandas

## Visualizations

The project includes multiple visualizations such as:

* Bar charts
* Distribution plots
* Correlation heatmaps
* Scatter plots
* Comparative analysis charts

## Key Insights

1. Countries with higher GDP generally report higher happiness scores.
2. Social support strongly contributes to overall happiness.
3. Nordic countries consistently rank among the happiest nations.
4. Corruption negatively affects happiness levels.
5. Freedom and life expectancy positively influence happiness.

## Conclusion

The analysis shows that happiness is influenced by multiple socio-economic factors. While GDP plays an important role, social support, freedom, and healthy life expectancy are also significant contributors. The findings highlight the importance of both economic and social well-being in determining national happiness.
